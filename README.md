# Rhino Plugins Collection

Gator, Peacock, and Roebuck handle the tedious parts of Rhino modeling at Spacio Techtonics.

We built this suite of plugins to solve the exact modeling and block-management headaches that cost us hours on real projects. If you've spent too much time manually editing instances or managing complex geometries, these tools might save you a lot of clicking.

## Plugins covered

| Plugin | Description |
|--------|-------------|
| Gator | Layout-space tools — detail naming, annotation, alignment, projected views |
| Roebuck | Area/length measurement, layer management, display modes, blocks |
| BlueWhale | OSM import, terrain, road generation, land-use colours |
